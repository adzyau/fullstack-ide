# 🚀 Fullstack Starter Template

A modern full-stack TypeScript template with PostgreSQL, GraphQL, Redis, and Docker support. Forked from [Sairyss/fullstack-starter-template](https://github.com/Sairyss/fullstack-starter-template).

---

## 🔧 Tech Stack

- **Frontend**: Next.js (React), TypeScript, TailwindCSS
- **Backend**: Node.js, GraphQL, TypeORM, Redis, PostgreSQL
- **Dev Tools**: Docker, ESLint, Prettier, Husky, Lint-staged, VS Code Settings
